Grocery Store Price Tracker

The grocery price tracker website will allow users to find the lowest food prices at grocery stores in Kelowna. Users will be able to track the prices of items across grocery stores and view general price trends. Commenting, and reporting prices on items will be restricted to registered users.

Skills: SQL, PHP, HTML, CSS, Wireframing, Relational Database Design

Game of Amazons AI Bot

Using the Minimax algorithm with Alpha-Beta pruning, and the minimum distance heuristic, a bot was created to play Game of Amazons. This project included researching and choosing your chosen algorithm, creating all Java classes and functions for game functionality, and implementing your chosen algorithm. Implementation included the functions needed to both send and receive moves within the game rules.

Skills: Object Oriented Programming, Java, Game theory algorithms

Capstone Project: Blur

Over the course of 8 months our team brought this software from a client concept to a finished, functional product. Blur is an open-source platform for easy and secure video-sharing. The purpose of our software is to allow admins (such as doctors, managers, professors, etc.) to request and receive videos from users. From this we can enable asynchronized communication which allows for more efficient communication, enhancing the experience for both the user and admin. The intention of our software is to provide an easy to use, secure solution to asynchronous video sharing. Blur was created using React and a large variety of AWS services including: Cognito, Rekognition, Amplify, Lambda, Appsync. While the website is no longer online due to the AWS hosting being connected to university accounts, a walkthrough of the product can be seen here: Video walkthrough

Skills: React, CSS, Front-end testing (Cypress), AWS Services, Wireframing (Figma)

Online Storefront

Using JSP, SQL, and AJAX an online storefront with both user and administrator portals. Some of the user features include: Product search, browse by category, product recommendations based on user, checkout, product detail pages, user account page, account creation (login/logout). Administrator features included: inventory display by warehouse, add/update/delete products, order and sales display. This project was dockerized and required our team to create the database and website design.

Skills: HTML, CSS, Java, JSP, SQL, Docker, Relational database design, AJAX

Brew Review - Android App

Brew Review is an app where users can discover and review new breweries and beer while keeping track of their favourite (and not so favourite) places to go. The user is able to add, edit, view, and delete reviews for breweries and beers. They can add and view the reviews through the brewery page, and delete and edit reviews through their user profile. They are also able to search the breweries and view their menu through the app. Finally, they are able to see all of their reviews (of beers and breweries) centralized in their user profile.

This is a front-end project, and therefore there is no functional backend. The intention of the project was to build a protoype, building from a paper prototype to a final high-fidelity prototype, incorporating user feedback from heuristic evaluations.

Skills: Android Studio, Java, UI Heuristics, Wireframing
